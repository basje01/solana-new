# Catalog Recommendations

Maps idea categories to specific repos, skills, and MCPs from the solana-new catalogs.

## How to Use

1. Identify the user's idea category from the list below
2. Recommend the listed repos, skills, and MCPs
3. Always check the actual catalogs for the latest entries — this file may be behind

## DeFi / Trading

**Repos**: jupiter-nextjs-example, raydium-swap-example, drift-examples
**Skills**: Jupiter Skill, DeFi Skill, Helius Skill
**MCPs**: Jupiter MCP, Helius MCP, DexScreener MCP
**Tip**: Start with Jupiter for swaps — best liquidity routing on Solana

## AI Agents

**Repos**: create-solana-agent, sak-telegram-bot, eliza-solana-plugin
**Skills**: Solana Agent Kit Skill, Phantom Connect Skill
**MCPs**: Solana Agent Kit MCP, Phantom MCP, Jupiter MCP
**Tip**: Solana Agent Kit is the standard — start there, extend with custom tools
